### pm2 run npm
```javascript
# pm2
$ pm2 start npm -- start

# npm
$ npm run start
```

### pm2 to node.js project
```javascript
$ npm install pm2
```

### pm2 command
```javascript
$ pm2 start app.js

$ pm2 stop     <app_name|id|'all'|json_conf>
$ pm2 restart  <app_name|id|'all'|json_conf>
$ pm2 delete   <app_name|id|'all'|json_conf>
```
